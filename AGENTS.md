# AGENT.md

## Projet

ParentZlite est une application mobile gamifiée qui aide les parents à progresser dans leur parentalité.

Le produit ne s'adresse PAS aux enfants.

Le produit s'adresse aux parents.

Le téléphone est utilisé uniquement par le parent.

---

## Vision

ParentZlite doit ressembler à :

* Duolingo
* Khan Academy Kids
* Headspace

Mais appliqué à la parentalité.

L'objectif n'est pas d'enseigner une théorie.

L'objectif est d'aider les parents à passer à l'action.

---

## Principe fondamental

Chaque interaction doit donner au parent le sentiment :

"Je progresse."

Jamais :

"Je suis un mauvais parent."

---

## Règles absolues

Ne jamais écrire :

* Mauvaise réponse
* Erreur
* Tu dois
* Tu aurais dû
* Mauvais parent
* Bonne réponse

Toujours préférer :

* Observons ensemble
* Cette réaction est compréhensible
* Cette approche peut aider
* Chaque petit pas compte
* Tu progresses avec ton enfant

---

## Mascotte

Nom :

Noxi

Noxi est :

* bienveillant
* curieux
* encourageant
* jamais moralisateur

Noxi n'est pas un expert.

Noxi apprend avec le parent.

---

## Design

Le design doit toujours respecter :

DESIGN.md

---

## Gamification

La gamification doit encourager.

Elle ne doit jamais punir.

Systèmes autorisés :

* XP
* niveaux
* badges
* streak
* graines
* tanière

Systèmes interdits :

* classement mondial
* compétition entre parents
* score négatif
* perte d'XP

---

## Développement

Priorité :

1. Mobile-first
2. Simplicité
3. Lisibilité
4. Rapidité
5. Performance

---

## Architecture
